Chiip是一个强大的AI编程助手工具，它提供了交互式命令行界面，能够帮助开发者完成各种编程任务。

## 使用说明

### 基本命令
- 直接输入编程需求进行对话
- `/help` - 显示帮助信息
- `/clear` - 清除对话历史
- `/config` - 显示当前配置
- `/set <key> <value>` - 设置配置项
- `exit` 或 `Ctrl+C` 两次 - 退出程序

- 创建一个简单的Web服务器
